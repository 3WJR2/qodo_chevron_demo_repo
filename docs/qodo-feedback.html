Embed anywhere

After publishing the repository with GitHub Pages (source: main, folder docs/), drop this iframe snippet into your site. The widget automatically fetches public data; provide a GitHub token if you need to read private repos or avoid rate limits. 

<iframe
  src="https://YOUR-USERNAME.github.io/qodo_chevron_demo_repo/qodo-feedback.html"
  title="Qodo PR feedback"
  style="width:100%;min-height:520px;border:0;"
  loading="lazy"
  allow="clipboard-write"
></iframe>

Publishing instructions follow the GitHub Pages + Jekyll workflow outlined in the official docs, so pushes to main automatically update this view.
